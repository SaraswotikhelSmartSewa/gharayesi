📁 Export / Import Config
Download your config as JSON to upload to GitHub, or import a backup.
Export

Download the current site-config.json. Upload it to your GitHub repo to update the live site. Also update the EMBEDDED_CONFIG in index.html for instant loading.

Import

Upload a previously exported config file to restore a backup.

🚀 How to Deploy to GitHub Pages
  1. Make your changes in admin, click Save All Changes
  2. Click Download site-config.json above
  3. Go to your GitHub repo → upload/replace site-config.json
  4. Open index.html in your repo → find EMBEDDED_CONFIG = {…} → replace the object with the downloaded JSON contents (minified)
  5. Your site updates instantly — no fetch dependency
  6. Important: Keep admin.html out of your public repo or password-protect it
