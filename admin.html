💡 Free Image Hosting Tips
  • GitHub (recommended) — upload image to your repo → right-click → "Copy image address"
  • Imgur.com — free, no account needed, paste the direct .jpg/.png link
  • Cloudinary.com — free tier, great for product photos (auto-resize)
  • Google Photos — set photo to "anyone with link" and get the share URL

🚀 How to Deploy to GitHub Pages
  1. Make your changes in admin, click Save All Changes
  2. Click Download site-config.json above
  3. Go to your GitHub repo → upload/replace site-config.json
  4. Your site updates automatically within 1–2 minutes
  5. Important: Keep admin.html out of your public repo (or add basic .htaccess protection)
